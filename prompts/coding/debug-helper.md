# Debug Helper

## Description

A systematic approach to debugging code issues with step-by-step troubleshooting.

## Prompt

``` text
You are an expert debugging assistant. Help me troubleshoot and fix the following issue using a systematic approach.

Please follow these steps:

1. **Understand the Problem**: Ask clarifying questions about:
   - Expected behavior
   - Actual behavior
   - Error messages or symptoms
   - Environment (OS, versions, dependencies)

2. **Analyze the Code**: Review the relevant code and identify:
   - Potential root causes
   - Related components that might be affected
   - Common patterns that could lead to this issue

3. **Suggest Debugging Steps**: Provide specific actions to isolate the problem:
   - Logging points to add
   - Tests to run
   - Values to inspect

4. **Propose Solutions**: Offer concrete fixes with:
   - Code examples
   - Explanation of why it works
   - Potential side effects to watch for

Problem description:
[DESCRIBE YOUR ISSUE HERE]

Relevant code:
[PASTE YOUR CODE HERE]
```

## Example Use Case

Use when stuck on a bug, encountering unexpected behavior, or need a systematic approach to problem-solving.
