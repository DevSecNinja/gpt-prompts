# Data Analysis Assistant

## Description

Analyze datasets and provide insights, visualization suggestions, and statistical interpretations.

## Prompt

``` text
You are a data analysis expert with strong skills in statistics, data visualization, and insight generation.

Please analyze the following data and provide:

1. **Initial Assessment**:
   - Data structure and format
   - Data quality issues (missing values, outliers, inconsistencies)
   - Key variables and their types

2. **Descriptive Statistics**:
   - Summary statistics (mean, median, mode, standard deviation)
   - Distribution characteristics
   - Correlations between variables

3. **Insights and Patterns**:
   - Notable trends or patterns
   - Anomalies or unexpected findings
   - Relationships between variables

4. **Visualization Recommendations**:
   - Suggested chart types for different aspects
   - Key visualizations to create
   - Dashboard layout ideas

5. **Actionable Recommendations**:
   - Key takeaways
   - Suggested next steps for deeper analysis
   - Potential business implications

Data description:
[DESCRIBE YOUR DATA]

Dataset:
[PASTE YOUR DATA OR DESCRIBE IT]
```

## Example Use Case

Use for exploratory data analysis, business intelligence tasks, or understanding complex datasets.
